## 🔧 **Quick Start**

### **Prerequisites**
- Python 3.8+
- Premium API Keys (Finnhub Pro recommended)
- $10K+ capital for Interactive Brokers integration

### **Installation & Setup**
```bash
# Clone repository
git clone https://github.com/ImaRogue/algorithmic-trading-platform.git
cd algorithmic-trading-platform

# Install dependencies
pip install -r requirements.txt

# Configure APIs
cp config/.env.example config/.env
# Edit config/.env with your API keys

# Run system launcher
python src/core/system_launcher.py

# Or run main trading engine directly
python src/core/trading_engine.py
```

### **System Validation**
```bash
# Run pre-market validation suite
python src/core/premarket_validator.py

# Test meme stock strategy
python src/strategies/meme_stock_momentum.py

# Full system scan
python project_scanner.py
```
